# Gutterball! Web App

### About
This project provides the front end for the *Gutterball!* app.

### Requirements
- yarn 2

### Setup
To set up the project, run:
```
$ yarn install
```

### Develop Locally
To begin development, run:
```
$ yarn start
```

### Run tests
Run all the tests with:
```
$ yarn test
```
